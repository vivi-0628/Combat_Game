# 程式介紹:
### === 爬塔遊戲 === 
### 一開始先選擇職業(1:戰士、2:盜賊、3:魔鬥士、4:弓箭手、...)
### 打怪時會有選項可選擇(1:普攻、2:技能、3:回復狀態)
### 每過一關能夠選擇通關獎勵or能力升級(小心寶箱怪)
### 不同職業有不同數值(血量、攻擊力、魔力...)
### 死亡時會返回第一關重新打起(選擇的能力值及等級將保留)
### 累計死亡10次將會進入特殊關卡
### 目標為打倒最終BOSS
